# 📄 PDF Merger Pro

A modern and feature-rich PDF management application built with Python and Tkinter.

PDF Merger Pro allows users to merge multiple PDF files into a single document, split PDFs into individual pages, reorder files before merging, and work with password-protected PDFs through an intuitive graphical interface.

---

## ✨ Features

### 📑 PDF Merging

* Merge unlimited PDF files
* Custom output filename support
* Automatic timestamp-based file naming
* Preserve original PDF quality

### ✂️ PDF Splitting

* Split PDF documents into individual pages
* Save pages automatically to a selected directory
* Progress tracking during operations

### 🔒 Security Support

* Handle password-protected PDFs
* Encrypted PDF detection
* Password verification before processing

### 🎨 Modern User Interface

* Beautiful Tkinter-based GUI
* Dark Mode and Light Mode
* Responsive layout
* Real-time status updates
* Progress bar visualization

### 📂 File Management

* Add multiple PDFs at once
* Reorder PDFs using Up and Down controls
* Remove selected files
* Clear entire file list
* Double-click to open PDF files

### ⚡ Performance

* Multi-threaded processing
* Non-blocking operations
* Smooth user experience during large PDF operations

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/pdf-merger-pro.git
cd pdf-merger-pro
```

### Install Dependencies

Using pypdf (recommended):

```bash
pip install pypdf
```

Or using PyPDF2:

```bash
pip install PyPDF2
```

---

## 🚀 Usage

Run the application:

```bash
python main.py
```

### Merge PDFs

1. Click **Add PDFs**
2. Select multiple PDF files
3. Arrange files in desired order
4. Choose output filename
5. Click **Merge PDFs**
6. Save and open the merged PDF

### Split PDF

1. Click **Split PDF**
2. Select a PDF file
3. Choose output folder
4. Wait for completion
5. Access generated page files
